employee-tracker-sql

## User Story

```AS A business owner
I WANT to be able to view and manage the departments, roles, and employees in my company
SO THAT I can organize and plan my business
```

## Requirements

SQL server running locally. Change any connection properties if needed within the `./db/connection.js` file. Default: localhost

## Usage

1. Install dependencies via `npm install`
2. `node index`
3. Application should be running, follow console prompts
